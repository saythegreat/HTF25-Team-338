Mechanics Project

Physics 1a • Professor Dr. Brown

Due Dec 20

Due Dec 20, 2024

Project Instructions

Build and analyze a simple machine that demonstrates mechanical principles learned in class.

Project Requirements:

  • Machine Type: Choose from: lever, pulley, inclined plane, or wheel and axle
  • Materials: Use household items or simple construction materials
  • Documentation: Include photos, measurements, and calculations
  • Analysis: Calculate mechanical advantage and efficiency
  • Presentation: 5-minute demonstration with explanation

Submission: Submit project report with photos and calculations as PDF.

Suggested Project Ideas

Lever System

Build a seesaw or balance scale using wood and weights

Pulley System

Create a simple lifting mechanism with ropes and pulleys

Inclined Plane

Build a ramp system to move objects with less force

Wheel & Axle

Create a simple cart or wheelbarrow mechanism
